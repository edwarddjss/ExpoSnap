# Contributing

## Setup

```bash
git clone https://github.com/YOUR_USERNAME/ExpoSnap.git
cd ExpoSnap
npm install
```

## Before Submitting

```bash
npm test
npm run typecheck  
npm run lint
npm run format
```

## Pull Request Checklist

- [ ] Tests pass
- [ ] Types check
- [ ] Lint passes
- [ ] Added tests for new features
- [ ] Updated README if needed
